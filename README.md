## À propos

Application mobile développée en partenariat avec le **Centre National de Veille Zoosanitaire de Tunisie (CNVZ)** pour révolutionner le suivi des vaccinations antirabiques.

### Fonctionnalités clés
- 📍 Suivi géolocalisé des chiens vaccinés
- 🗺️ Cartographie des zones à risque
- 📊 Statistiques en temps réel
- 💉 Gestion des campagnes de vaccination

---

## Tech Stack

**Mobile** : React Native  
**Backend** : Node.js, Express.js  
**Database** : MongoDB  
**Maps** : React Native Maps

## Installation

```bash
git clone https://github.com/KRIDAmani/WoofIt_MoveIt-App.git
cd WoofIt_MoveIt-App
npm install

# iOS
npx react-native run-ios

# Android
npx react-native run-android
```
